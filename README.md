# 📊 Data Explorer

A personal data science portfolio project built with Python, pandas, and seaborn.

## 📁 Notebooks

| Notebook | Dataset | Description |
|---|---|---|
| 01_salaries.ipynb | DS Job Salaries | Analysis of data science salaries by role, experience, and year |

## 🛠️ Tech Stack

- Python 3.13
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## 🔍 Key Findings

- **Experience level drives salary more than job title**: Executive-level roles earn
  roughly 3-4x more than entry-level positions across all job categories
- **Salaries grew significantly between 2020 and 2022**: average compensation in
  data science roles increased substantially year-over-year, reflecting high market demand
- **Location matters enormously**: US-based roles dominate the top salary brackets,
  while equivalent roles in India, Brazil or Eastern Europe pay 5-10x less in USD
- **ML Engineer and Data Architect are the highest-paying specializations**,
  consistently outperforming generic Data Analyst roles at the same experience level
